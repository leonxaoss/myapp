Зарегистрироваться на https://unsplash.com/developers, создать свой application, и использовать их REST API для того, чтобы отобразить на своей страничке некоторые изображения. Для использования REST API с сайта применить https://github.com/unsplash/unsplash-js Обязательное условие - использовать Promise. Необходимо использовать Promise для поучения данных с сервера, и создавать элемент изображения на своей странице с прелоадером и обязательно отлавливать ошибки. Результат выполненного задания предоставить ссылкой на git репозиторий. Дополнительным плюсом будет выполнение задания с помощью JS-фреймворка или использование Gulp/Webpack/Grunt etc..
